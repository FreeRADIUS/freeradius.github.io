164 /** Retrieve errors from the last query operation
165  *
166  * @note Buffers allocated in the context provided will be automatically freed. The driver
167  * should not free these buffers explicitly.
168  * @note If the driver uses its own buffers to aggregate messages, they should be cleared
169  * on sql_query_finish, and after each call to sql_error, to prevent the same messages
170  * being printed multiple times.
171  *
172  * @param[in,out] ctx to allocate any buffers required. If static buffers are provided by the
173  * driver they need not be talloc_strdupd, just write the pointer to those buffers to the
174  * .msg field of a sql_log_entry_t element.
175  * @param[out] out a pre-allocated array of log entries to fill. Need not be NULL terminated.
176  * @param[in] outlen Number of log entries available for populating. Do not write to index
177  * out[outlen] or higher.
178  * @param[in] handle to retrieve errors from.
179  * @param[in] config of the SQL instance.
180  * @return
181  * 0 - If no error messages are available.
182  * >0 - Number of log entries
183  */
