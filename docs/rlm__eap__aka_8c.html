static int mod_section_compile ( eap_aka_sim_actions_t *  actions,
CONF_SECTION *  server_cs 
)
static

Compile virtual server sections.

Called twice, once when a server with an eap-aka namespace is found, and once when an EAP-AKA module is instantiated.

The first time is with actions == NULL and is to compile the sections and perform validation. The second time is to write out pointers to the compiled sections which the EAP-AKA module will use to execute unlang code.
