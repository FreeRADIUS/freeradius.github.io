fr_dict_attr_t **  out,

)

Create a dictionary attribute by name embedded in another string.

Find the first invalid attribute name char in the string pointed to by name.

Copy the characters between the start of the name string and the first none fr_dict_attr_allowed_chars char to a buffer and initialise da as an unknown attribute.

Parameters
[in]ctxTo allocate unknown fr_dict_attr_t in.
[out]outWhere to write the head of the chain unknown dictionary attributes.
[in]parentAttribute to use as the root for resolving OIDs in. Usually the root of a protocol dictionary.
[in]namestring start.
Returns
  • <= 0 on failure.
  • The number of bytes of name consumed on success.
