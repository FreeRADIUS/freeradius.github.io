◆ linkset_t18_cb()

static void linkset_t18_cb ( void *  _set)
static

T18 is used on linkset restart when we're an STP (Signalling Transfer Point)

It's shorter than the T20, and indicates a period where we listen, but don't rebroadcast TFAs from adjacent connections.

The time between T18 and T20 is when we rebroadcast link advertisements we've received, to adjacent SPs.
