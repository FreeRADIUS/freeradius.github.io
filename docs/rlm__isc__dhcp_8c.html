struct rlm_isc_dhcp_info_s

Holds information about the thing we parsed.

Note that this parser is forgiving. We would rather accept things ISC DHCP doesn't accept, than reject things it accepts.

Since we only implement a tiny portion of it's configuration, we tend to accept all kinds of things, and then just ignore them.
