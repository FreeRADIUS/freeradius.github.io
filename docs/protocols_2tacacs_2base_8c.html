Return how long a TACACS+ packet is.

Note that we only look at the 12 byte packet header. We don't (yet) do validation on authentication / authorization / accounting headers. The packet may still be determined later to be invalid.

Parameters
bufferto check
buffer_lenlength of the buffer
Returns
>0 size of the TACACS+ packet. We want. MAY be larger than "buffer_len" <=0 error, packet should be discarded.
